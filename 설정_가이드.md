# 설정 파일 가이드

## 필수 항목

OHLCV 데이터 조회를 위해 **반드시** 설정해야 하는 항목:

### 실전투자 사용 시
- `my_app`: 실전투자 앱키 ✅ **필수**
- `my_sec`: 실전투자 앱시크릿 ✅ **필수**
- `my_acct_stock`: 증권계좌 8자리 (선택사항이지만 권장)

### 모의투자 사용 시
- `paper_app`: 모의투자 앱키 ✅ **필수**
- `paper_sec`: 모의투자 앱시크릿 ✅ **필수**
- `my_paper_stock`: 모의투자 증권계좌 8자리 (선택사항이지만 권장)

## 선택 항목

다음 항목들은 **선택사항**입니다. 해당 기능을 사용하지 않으면 설정하지 않아도 됩니다:

- `my_htsid`: HTS ID
  - **용도**: 체결통보, 나의 조건 목록 확인 등
  - **OHLCV 조회**: 불필요 ❌
  - 설정하지 않으면 빈 문자열로 처리됩니다

- `my_acct_future`: 선물옵션계좌 8자리
  - **용도**: 선물/옵션 거래
  - **OHLCV 조회**: 불필요 ❌ (주식만 조회하는 경우)
  - 설정하지 않으면 빈 문자열로 처리됩니다

- `my_paper_future`: 모의투자 선물옵션계좌 8자리
  - **용도**: 모의투자 선물/옵션 거래
  - **OHLCV 조회**: 불필요 ❌
  - 설정하지 않으면 빈 문자열로 처리됩니다

## 설정 예시

### 최소 설정 (OHLCV 조회만 하는 경우)

```yaml
# 실전투자
my_app: "실제_앱키"
my_sec: "실제_앱시크릿"

# 모의투자
paper_app: "실제_모의투자_앱키"
paper_sec: "실제_모의투자_앱시크릿"

# 계좌번호 (선택사항)
my_acct_stock: "68321178"

# 계좌상품
my_prod: "01"
```

나머지 항목은 설정하지 않아도 됩니다!

## 주의사항

1. **예제 텍스트 제거**: "여기에 입력", "8자리" 같은 예제 텍스트는 자동으로 무시됩니다.
2. **따옴표**: YAML 파일에서 값은 따옴표로 감싸도 되고 안 해도 됩니다.
3. **빈 값 처리**: 설정하지 않은 항목은 자동으로 빈 문자열로 처리되어 오류가 발생하지 않습니다.
